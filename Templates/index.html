Project Overview

This project applies deep learning to automatically detect and classify defects in screws. Using a ResNet50-based model, it supports both image and video analysis. Users can upload individual images for classification or entire videos for frame-by-frame detection. The system highlights predictions directly on the media and displays results in an elegant web interface. This helps streamline quality control by providing quick and reliable defect detection.
